
# Student Marks Analysis - Power BI Dashboard

## Overview
This Power BI dashboard provides insights into student performance across subjects, helping educators and stakeholders analyze trends and make data-driven decisions.

## Features
- **Average Marks per Subject:** Visualizes subject-wise performance.
- **Top 5 Student Marks:** Highlights the highest-scoring students.
- **Subject-wise Performance:** Tracks performance trends across subjects.
- **Class Filter (Slicer):** Enables users to filter data by class for focused analysis.

## Technologies Used
- **Power BI**
- **Excel / SQL Database** (for storing student marks data)
- **DAX & Power Query** (for calculations and transformations)
